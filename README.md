# Instagram High-Res Image Downloader (ig-downloader)

인스타그램 브라우저 환경에서 썸네일(Thumbnail)이 아닌 **최고 해상도(High-Res)의 원본 이미지**를 다운로드 할 수 있게 해주는 빠르고 가벼운 크롬 확장 프로그램입니다.

---

## 🌟 주요 기능 (Features)

* **원본 해상도 다운로드:** 인스타그램 렌더링에 사용되는 블러(Blur) 처리된 Placeholder 이미지를 무시하고, `srcset`을 감지하여 최고 화질의 원본 이미지 URL만을 추출하여 다운로드합니다.
* **캐러셀(Carousel) 완벽 지원:** 여러 장으로 이루어진 슬라이드 포스트에서도 개별 이미지 컴포넌트에 다운로드 버튼이 종속되어, 넘기면서 원하는 사진만 즉시 다운로드할 수 있습니다.
* **불필요한 그리드 무시:** 사용자의 프로필 피드 등 썸네일 그리드에서는 동작하지 않고, 실제 상세 게시물(`<article>`) 내부 및 피드 화면에서만 스마트하게 동작합니다.
* **직관적인 UI / UX:** 이미지에 마우스를 올릴 때만(Hover) 우측 상단에 깔끔한 다운로드 버튼이 표시됩니다. 클릭 즉시 직관적인 체크 아이콘(✅)을 보여주며 피드백을 제공합니다.
* **정돈된 파일명:** 파일을 내려받을 시 지저분한 임의의 해시값이 아닌, 체계적인 `ig_highres_시간.jpg` 형식으로 저장되어 관리하기 쉽습니다.

---

## 🛠 설치 방법 (설치 / 셋업 메뉴얼)

이 프로그램은 로컬 컴퓨터에서 **개발자 모드**를 통하여 곧바로 설치할 수 있습니다.

1. 로컬 환경에 해당 소스 파일 폴더 (`ig-downloader`) 전체를 다운로드하거나 클론(Clone)합니다.
2. 크롬 브라우저를 열고 주소창에 `chrome://extensions/` 를 입력하여 접속합니다. (우측 상단 톱니바퀴 > 확장 프로그램 관리)
3. 화면 **우측 상단에 있는 [개발자 모드]** 토글 스위치를 활성화(ON) 합니다.
4. 화면 좌측 상단에 나타나는 **[압축해제된 확장 프로그램을 로드합니다]** 버튼을 클릭합니다.
5. 소스코드가 담겨져있는 파일인 `ig-downloader` 폴더를 통째로 선택합니다.
6. 목록에 퍼즐 모양 아이콘을 가진 `Instagram Image Downloader`가 나타나고 스위치가 파란색(ON)으로 켜져 있으면 설치 완료입니다!

---

## 🚀 사용 방법 (How to use)

1. 크롬을 통해 [Instagram.com](https://www.instagram.com)에 접속합니다.
2. 메인 피드의 게시물을 구경하거나, 포스트 클릭을 통해 상세 팝업(Modal)창을 띄웁니다.
3. 소장하고 싶은 **이미지 위로 마우스를 올립니다.**
4. 이미지 우측 상단에 어두운 반투명 색상의 **[⬇️ 다운로드] 버튼**이 나타납니다.
5. 버튼을 클릭하면 `ig_highres_YYYY-MM-DD-HH-MM-SS.jpg` 의 이름으로 고화질 사진이 다운로드 폴더에 자동 저장됩니다!

---

## ⚠️ 주의 사항

* **비공개/보안 이슈:** 인스타그램 자체의 개인정보/스크래핑 보안 패치가 대규모로 진행될 경우 버튼이 일시적으로 보이지 않을 수 있습니다.
* **동영상 지원 환경:** 본 확장 프로그램은 릴스 등의 동영상 포맷(`<video>`)이 아닌 **일반 이미지(`.jpg` 등)** 추출을 목적으로 제작되었습니다.
* **저작권 주의:** 다운로드한 이미지들의 사용 권한 및 저작권은 각 인스타그램의 원작자에게 있습니다. 개인 소장 용도로만 사용하시고, 무단 배포 등에 주의해 주시기 바랍니다.
